Designed to replicate the club's previous website which was hosted on a now redundant platform, the South Notts ARC website is a simple responsive website. It is available in desktop and smartphone browser formats. The colour scheme aims to make reading the content easier for those with eyesight difficulties. The club administrators access a set of pages hidden from normal view, to update the various categories of information. The new website has been well recieved.

Data for the site is held in a MySQL database, which I wrote php code to manage, including creating the tables, creating queries, select ,insert and updating. As access to this data is restricted by the host server, there is also a better level of security than bespoke files would provide.
